ScratchAddons

Welcome to the ScratchAddons website! ScratchAddons is a browser extension project started by World_Languages, it aims to collect, archive and put into use userstyles and userscripts. Currently, ScratchAddons is being developed on GitHub by several members of the scratch community. Once development on the project is close to done, this addon will be released into the chrome extention store and (hopefully) merged with griffpatch's Scratch Messaging Addon.
