How to download ScratchAddons

  1. Go to the GitHub repository for the addon.
  2. Find the "Clone of Download" button:
  3. Click the button and select "Download ZIP":
  4. Unzip the file to a new folder and go to your Chrome Extensions section and turn on developer mode:
  5. Click "Load unpacked" and select the folder with the extension:
  6. Congrats! You now have ScratchAddons installed! To get started click on the 3 dots next to the extension in the toolbar and select "Options". You will then be taken to a settings page where you can turn on and off the addons!
